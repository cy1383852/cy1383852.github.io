html5 新属性

HTML5

标签： html5 新属性

[TOC]

概念

  1. html的第5个版本。
  2. 包含html.css，js等这里的HTML5是1 。

HTML5增添了许多新标签，更具有语义化。追加了本地数据库等Web应用功能

HTML5新增常用标签

HTML5新增常用标签 标签作用
articl 定义文章：论坛帖子、博客文章、新闻故事、品论
aside 定义和页面内容关联度较低的内容
datalist 定义列表项
detauls 定义展开详细内容 有属性open summary定义标题内容
figure 定义一个和文档流有关的图例（独立的流内容） figcaption 代表一个图例的说明
section 定义文档中的某个区域
header 定义页面或章节的头部
footer 定义页面或章节的尾部
main 定义文档中主要部分和重要部分
nav 定义导航部分
mark 定义有标记的文本
time 代表时间（但并没有什么特殊）
embed 代表一个嵌入的外部资源，如应用程序或交互内容
progress 代表进度条
output 代表输出内容的标签

新增通用属性

属性 意义

hidden 隐藏元素

DragDrop API

处理步骤

  • 定义可移动目标
  • 定义被拖动的数据，可能为多种不同格式
  • 允许设置拖拽效果
  • 定义放置区域
  • 在drop发生的时候做一些事情

dataTransfer

  • 用来保存被他拖动元素的数据
  • 可以保存一项或多项数据，一种或多种的数据类型
  • 注意：在所有的拖动事件属性dataTransfer都是可用的，但不能单独创建
  • 方法 setData（） 设置数据；getData（）获取数据

拖放事件

  • dragstart ：拖拽开始（元素默认都是可以拖拽的，或设置属性draggable = “true”【被拖拽元素设置】
  • dragend ： 拖拽结束 【被拖拽元素设置】
  • dragenter ： 拖拽时，鼠标进入（通常设置进入后效果）
  • dragover ：拖拽时，鼠标进入后反复执行，（设置目标元素允许接受拖放元素）//e.preventDefault（）
  • dragleave ： 拖拽时，鼠标离开
  • drop ：

File API

  • files：读取上传的文件相关信息
  • 是个数组，存放所有选择的文件 多选需设置 multiple 属性
  • length 获取有多少个文件

File对象

  • lastModified 最后修改时间
  • lastModifiedDate 最后修改时间对象
  • name 文件名称
  • size 文件大小 字节
  • type 文件类型 获取具体类型字符串拆分【type.split(“/“)[1]】

注意使用JQ如何获取

  1. $(‘#file’).get(0).files
  2. $(‘#file’)[0].files
